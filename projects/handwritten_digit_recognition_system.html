Project information

  • Title: Handwritten Digit Recognition System
  • Domain: Deep Learning
  • Project date: November, 2022
  • Project URL: Handwritten-digit-recognition-system

Portfolio Detail

In a collaborative effort with a team of four, we designed and implemented a simple web application using Python and Flask to recognize handwritten digits. The application accepts an image of a handwritten digit as input and accurately predicts the number using a machine learning algorithm, achieving an impressive accuracy rate of 98.76%.

This project not only showcased our ability to develop a functional web application but also highlighted our expertise in applying machine learning techniques to real-world problems, enhancing the user experience through accurate and efficient digit recognition.
